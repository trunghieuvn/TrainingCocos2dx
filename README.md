Tool Build: 
## Tools Build Win32 & Android 
 - Version Cocos2d-x: 3.16 
 - Android SDK 
 - Android NDK: r10e 
 - ANT 1.9 
 - Visual Studio 2013 

## Command build
- Win32: 
  ```
   cocos compile -p win32 
  ```
- Android: 
  ``` 
  cocos compile -p android --android-studio 
  ```
## Script Build: Setup, Build win32, Build Android sử dụng 

  ```
    00_Make_Menu.bat
  ```
